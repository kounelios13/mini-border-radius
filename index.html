Here you will learn how to install and setup miniRadiusGenerator in your web application.
First you need to include the relevant js file to your html code like this:
< script src="path/to/miniRadiusGenerator">< /script>.Also make sure you include the relevant css file:
< link rel="stylesheet" href="path/to/miniRadius"/>. Now let's go to the interesting part.Each border radius generator(Yeap right.You can have as many generators as you want) lives in it's own div.To initiate and activate a generator you have to create a new instance of the Generator class and provide it with the id of the host div.You do it like this:
var generator;
$(document).ready(function(){ generator = new Generator("#sample");
//for some reason you have to initiate the generator
//inside the ready() function.Hope to have this fixed in a future release though.
});
The above code will initiate a tiny generator tha will be injected in your page.But what happens when you try to move a slider from the generator that has been created?And the answer is pretty obvious.NOTHING!!!That's because we haven't attach any event listener to our host div.To do this we have to write the following code inside the ready() function; $('#sample').on('mousemove',"input[type='rane']",function(){generator.activateGenerator()});
This line of code will activate the true functionality of your generator.You will notice there is a green text area in your screen which is dynamically updated from your generator.Each generator has its own text are to ouput the code generated

So basically this is a basic installation of our generator.But you see there are some things you might don't like.For example: I want a larger generator or I want to set a different backgound or different max value.How can I do that?The answer is simple.Your generator has an init method which takes a list of 4 elements as an arguments:height,width,max_value,background_color.You can call it like this:
generator.init([height,width,max_value,background]);. For example you can write the following code and press the red button to see what happens:
generator.init(["15em","15em",200,"gold"]); Click me and let the magic happen
