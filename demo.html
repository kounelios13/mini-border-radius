What is miniBorderRadius?

miniBorderRadius is a plugin written mostly in jQuery which can add a border radius generator in your web page.

How to install?

It's pretty simple.All you have is to invlude the following files in your html:
<script src="path/to/miniBorderRadius.js"/>
<link rel='stylesheet' href='path/to/miniBorderRadius.css'/> There is also a minified js version in the same folder called "miniBorderRadius.min.js".

Dependencies?

This plugin is written in jQuery but is also using some bootstrap components so here are what you need in your project:

  • jQuery
  • twittwer bootstrap

So now what?

Now that you have included all relevant files there are a few things you need to do. Each generator(yeap you can have unlimited generators in a single page) lives in it's own div.To initiate and activate a generator you have to create an instance of the Generator class and provide it with th id of the host div like this:
var generator;
$(document).ready(function(){
generator = new Generator("#sample");
//for some reason you have to initiate the generator
//inside the ready() function.Hope to have this fixed in a future release though.
});
 Instead of using the id of the host div as an argument you can pass an array of default options like this:
generator=new Generator(["#sample","10em","10em",200,"maroon"]); Now let me explain what this list is.The first item of the list is the id of the host div.The second and the third is the height and the width of the generetor.The fourth value is the max value for the sliders of the generator.And the fifth item is the background of the generator.There is also a third constructor which consists of 2 parameters.The first parameter can be a string or a list as the previous 2 constructors.The second parameter is the new object that will be replacing your current generator output object.For example: generator=new Generator("#sample","#image"). Generator methods

Each generator has some basic methods.Here are:

Toggle methods
.activateGenerator()
This is the main function that activates the generator.Chainable:Yes
.deactivateGenerator()
This function resets the generator and then deactivates it so be sure you got the code from the generator before deactivating it.Chainable:Yes
.reset()
Reset the values of the generator.Chainable:Yes
.replaceObject(object,restrict_size)
Replace the default object where border radius is applied with another one.If you want to keep the new object to have the same size as the previous one pass true as the second parameter in the function .Chainable:Yes
.setSize(height,width)
Does what it says :p
.setMax(value)
Set the maximum value for all the sliders inside a generator.Chainable:Yes
.setBackground(color)
set the background color of a generator.Chainable:Yes
.init(options)
This function accepts a list of 4 default options:height,width,max_value,default_background.Chainable:Yes
.getCode()
Gets the code generated from a miniRadiusGenerator.Chainable:No
.getOptions()
Returns a list with the default options of the generator.Chainable:No
.getId()
Returns the id of the host div.Chainable:No
.addToFavourites()
Adds the curent code produced by the generator into a list.Chainable:Yes
.getFavourites()
Returns favourites list.Chainable:No
.removeFavourites()
Removes favourites.Chainable:Yes
.toggleFavourites()
Toggle the favourites list created by the current generator.Chainable:Yes
.enablePreview()
When you execute this method you are able to preview any border radius in your current favourites list just by clicking it.Chainable:Yes

If you don't remember wheter a function is chainable or not you can call isChainable(name) and pass teh name of the function you want to check.You will get a true or false.For example the following: console.log(isChainable("init")) ; will log true because init function is chainable.You will get the same output if you write console.log(isChainable(".init")); or console.log(isChainable(".init()"));.
